# Siloq Dashboard

A clean Next.js project with TypeScript and Tailwind CSS setup.

## Quick Start

```bash
# Install dependencies
npm install

# Development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Project Structure

```
src/
├── app/
│   ├── globals.css
│   ├── layout.tsx
│   └── page.tsx
├── components/
│   └── ui/
└── lib/
    └── utils.ts
```

## Stack

- **Framework** — Next.js 15, React 18
- **Language** — TypeScript
- **Styling** — Tailwind CSS
- **Linting** — ESLint

## Available Scripts

```bash
npm run dev      # Start development server
npm run build    # Production build
npm run start    # Start production server
npm run lint     # Run ESLint
```

## Branches

- **main** — Clean Next.js project (this branch)
